<h2>DESCRIPTION</h2>
To use this Program in Controlwave Designer. 
Create a new Structure Text Program, Right Click Logical POUs>>Insert>>Program, </br> 
-   Select Language " SFC ".
</br>
-   Copy <# Code #> Block
</br> 
-   Create Signals from the <# Signals #> block
